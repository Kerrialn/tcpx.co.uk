Project Upgrade
A project upgrade often involves upgrading from a programming language and/or upgrading a framework version. We take the following steps to ensure a safe and consistent upgrade.
Analysis
We sign a Non-disclosure agreement (NDA) so you know your code is in safe hands.
We run automated analysis programs to find issue in your codebase.
We deliver, for a one-time pre-payment of 5,000 - 8,000 EUR, a PDF report of found issues and a timeline for fixing them.
Upgrade
We will work in parallel with your team for 20 - 40 hours per week implementing fixes to your codebase.
We start with the smallest and easiest issues, we work on a start one - finish one basis, this means we create a change and merge it immediately, no stale branches.
We can schedule once a month meetings to discuss progress, blockers and adapt strategy if need be.
How much does it cost?
5,000 - 8,000 EUR one-time payment for the analysis report
120-150 EUR per hour / invoiced weekly or monthly
